# Resume Flow - AI Resume Builder Authentication System

A premium, modern split-screen authentication portal engineered for **Resume Flow**—an intelligent, AI-powered resume builder website. This subsystem provides a high-converting onboarding experience, featuring an interactive user form workspace on the left and a dynamic dashboard feature preview on the right.

## 🚀 Features

- **Split-Screen Layout:** Mirroring modern SaaS designs, providing an immersive brand experience while keeping the focus on user registration and login.
- **100% Semantic HTML & CSS Architecture:** Formatted completely without using `<div>` or `<span>` elements to achieve immaculate SEO indexability, web standards, and accessibility.
- **Embedded Password Toggle:** A custom, integrated JavaScript password visibility eye switch to minimize login errors.
- **Social Connectors:** Polished template configurations for single-sign-on (SSO) authentication via Google and Apple.
- **Fully Responsive Matrix:** Fluid CSS layouts that gracefully collapse the right visual marketing view on smaller viewports (mobile/tablet devices).

🤖 The AI Platform Connection Flow
This authentication subsystem serves as the secure entry gateway into the main Resume Flow AI Engine:

User Sign Up / Log In: The user creates an account securely through the semantic form layouts.

Session Verification: Once authenticated, the form directs the user to the central Resume Flow Workspace Dashboard.

AI Resume Engine: Inside the dashboard, the user can input job titles and skills, leveraging AI text generation to instantly compose tailored summaries, bullet points, and ATS-optimized cover letters.

Data Persistence: User credentials mapped via these authentication sheets ensure their generated resumes, layout templates, and edit histories remain safely synced to their individual cloud profiles.
